にこぐらふの概要

主な機能

ニコニコ動画の動画IDをPOSTするとその動画のコメント情報を引っ張ってくるためのジョブがキューに追加されます．裏側では，キューに追加されたのを感知して対応する動画につけられたコメントに関する情報をひっぱってきて，その中に含まれている，コメントがつけられた再生時間軸上の位置やコメントの内容などを基にテキトーに計算を行い，その結果を横軸を再生時間，縦軸をある指標の大きさとしたグラフとして表現し，画像に出力します．

にこぐらふの概要

主な機能

  • 対象の動画IDをキューに追加
  • コメントを取得
  • 文字「w」についてほげほげ
  • グラフ生成

にこぐらふの概要

動画とグラフを同時表示

にこぐらふの概要

その他の機能（LL Nagoya 後に少々）

  • 動画の全文検索
    • ※ただし登録されている動画に（ｒｙ
  • タグによる検索
    • 全文検索とあまりかわｒ（ｒｙ
  • ニコニコ動画のRSS情報から自動的にPOST
    • 誰もPOSTしてくれないのでむしゃくしゃしてやった

にこぐらふの概要

だいたいの構成

  • nginx + PSGI Appllication
  • MySQL + Mroonga (groonga storage engine)
    • mroonga-1.1.0 / groonga-1.2.8 @ 2011-11-29
  • イベント駆動プロセス
    • コメント取得
    • 計算ごにょごにょ
    • グラフ生成（メモリ食い！！）
    • TwitterへPOST
  • さくらのVPS 512
    • スワップ多発ｗ

にこぐらふの概要

だいたいの構成

  • Perl
    • Amon2 - 薄いWAF
    • Teng - 薄いO/Rマッパ
    • AnyEvent::Filesys::Notify - ディレクトリを監視してイベント駆動
    • Chart::Clicker - グラフ生成
    • XML::LibXML
    • LWP::UserAgent
    • OAuth::Lite
    • Net::Twitter::Lite - Twitter
